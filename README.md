# rpi-fan-control (archived C version)

**This version (branch [archived-c](https://github.com/fybx/rpi-fan-control/tree/archived-c)) is archived. Please see Rust rewrite at branch [main](https://github.com/fybx/rpi-fan-control).**

A simple and highly-optimized C program to turn Raspberry Pi's fan on and off.

See `build.sh` for building details, requires [joan2937/pigpio](https://github.com/joan2937/pigpio) library to function. Use `deploy_service.sh` to deploy SystemD service and timer.
